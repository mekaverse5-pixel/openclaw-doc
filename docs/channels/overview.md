# 消息通道概述

OpenClaw 支持多种消息通道，让你可以使用自己喜欢的聊天应用与 AI 助手交互。

## 支持的通道

| 通道 | 状态 | 说明 |
|------|------|------|
| Telegram | ✅ 已支持 | 最推荐的通道 |
| Discord | ✅ 已支持 | 适合社区使用 |
| WhatsApp | ✅ 已支持 | 需要手机在线 |
| Slack | ✅ 已支持 | 适合团队协作 |

## 快速配置

每个通道的配置步骤类似：
1. 获取通道的 API 密钥或机器人令牌
2. 在 OpenClaw 中配置通道
3. 设置访问权限

详细配置请参考各个通道的具体文档。
