# 浏览器自动化工具

AI 可以控制浏览器进行操作。

## 功能

- 打开网页
- 截图
- 填写表单
- 点击元素
- 滚动页面
- 获取页面内容

## 启用

```json
{
  "tools": {
    "browser": {
      "enabled": true
    }
  }
}
```

## 使用示例

```
打开 https://example.com
截图
点击按钮 "提交"
填写输入框 "用户名" 为 "test"
```

## 注意事项

- 首次使用需要安装浏览器驱动
- 建议使用 Chromium 内核浏览器
- 需要显示服务器环境（无头模式）
