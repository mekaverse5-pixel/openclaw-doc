# 配置指南概述

OpenClaw 提供丰富的配置选项，让你根据需求定制 AI 助手。

## 主要配置项

| 分类 | 说明 |
|------|------|
| Gateway | 核心服务配置 |
| Tools | 工具权限管理 |
| Security | 安全和访问控制 |

## 配置文件位置

主配置文件：`~/.openclaw/openclaw.json`

查看当前配置：
```bash
openclaw config show
```

## 验证配置

```bash
openclaw doctor
```
