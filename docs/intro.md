# OpenClaw 中文文档

欢迎使用 OpenClaw！

OpenClaw 是一个强大的开源 AI 助手框架，支持多种消息通道和 AI 模型。

## 特性

- 🤖 **多模型支持** - OpenAI、Claude、MiniMax 等
- 💬 **多通道集成** - Telegram、Discord、WhatsApp 等
- 🔧 **丰富工具** - 文件操作、终端命令、浏览器自动化
- 🔒 **安全可控** - 细粒度权限控制
